# 路线图 | Roadmap

目标：5000个深度知识节点，1000位共建者，一个真正活着的知识宇宙。

Goal: 5000 deep knowledge nodes, 1000 co-builders, one truly living knowledge universe.

---

## Phase 1: 基础 | Foundation (Now)
- [x] Core knowledge graph (52 nodes, 8 disciplines, 109 connections)
- [x] Interactive 3D visualization
- [x] Depth content exemplar (feedback loops)
- [x] Open source on GitHub
- [ ] 第一批20位共建者 | First 20 companions
- [ ] 10个带深度内容的节点 | 10 nodes with depth content

## Phase 2: 生长 | Growth (Next 3 months)
- Target: 100 active companions
- 100 nodes with full depth content
- Weekly knowledge node releases
- Community review process established
- Bilingual docs (Chinese + English)

## Phase 3: 网络 | Network (Year 1)
- Target: 1000 companions
- 500+ depth nodes
- Cross-community partnerships
- Educational use cases (curriculum integration)
- Translation system for global access

## Phase 4: 生物 | Living (Ongoing)
- Real-time knowledge integration from open sources
- AI-assisted connection discovery
- Self-correcting mechanism for outdated information
- Autonomous growth within human-guided boundaries
